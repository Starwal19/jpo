Easy - Introduction :

Ce premier niveau a pour objectif de vous familiariser avec steghide.

Vous avez devant vous trois images dont l'une d'elle contient un message caché.

Vous avez comme objectif de touver qu'elle image contient ce message et de l'extraire.

Un mot de passe est nécéssaire, vous allez devoir le deviner.

Il s'agit de l'un des pires mot de passe à utiliser. (Google est votre ami)

Bonne chance.

Easy - Indices :

Indice 1 : La commande 'steghide info <image>' peut vous servir à trouver quelle image contient le fichier caché.

Indice 2 : La commande 'steghide extract -sf <image>' peut vous servir à extraire un fichier d'une image.

Indice 3 : Cherchez sur le WEB quel est le pire mot de passe, vous le trouverez facilement.
